---# LobsterAI 开发者文档

LobsterAI 是一个完全开源的 AI Agent 项目。源代码、开发说明和最新实现都托管在官方 GitHub 仓库中。为了避免官网文档与实际代码不同步，本页面不再维护独立的开发者文档；请以 GitHub README 和仓库内容为准。

## 开源协议

LobsterAI 采用 MIT 协议。你可以在遵守 MIT 协议的前提下学习源码、复用代码、二次开发，并将它集成到自己的工作流或产品中。

## 去哪里看开发者文档

官方 GitHub 仓库：

[https://github.com/netease-youdao/LobsterAI](https://github.com/netease-youdao/LobsterAI)

开发、运行、构建、贡献等说明请查看官方 README：

[查看 GitHub README](https://github.com/netease-youdao/LobsterAI#readme)

## 欢迎参与

如果 LobsterAI 对你有帮助，欢迎在 GitHub 上给项目一个 Star，也欢迎分享给需要的朋友或团队，或者 Fork 仓库进行二次开发。

你可以通过这些方式参与：

- 提 Issue 反馈问题或需求
- 提 Pull Request 改进代码或文档
- Fork 后做自己的定制版本
- 将项目分享给更多开发者和 AI Agent 用户

开源项目的生命力来自社区。欢迎一起把 LobsterAI 做得更好。
