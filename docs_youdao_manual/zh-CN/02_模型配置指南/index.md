## 📖 什么是API Key?

### 基本概念

想象一下:
- **AI模型**是一个超级聪明的助手,运行在服务商的云端服务器上
- **API Key**是服务商给你的专属访问密钥
- **TopVanAI**是帮你连接和使用这些AI模型的工具

### 使用流程

1. 在AI服务商网站注册账号
2. 获取该服务商提供的API Key(密钥)
3. 在TopVanAI中配置API Key
4. 即可在TopVanAI中使用该模型

### 费用说明

- **TopVanAI本身免费**,我们不收取任何费用
- AI模型服务由各服务商提供,费用直接支付给服务商
- 就像微信(免费)和滴滴打车(付费)的关系,微信是工具,打车服务需要单独付费

## 📋 配置流程概览

所有模型的配置流程基本相同:

```
1️⃣ 注册账号 → 2️⃣ 实名认证 → 3️⃣ 获取API Key → 4️⃣ 账户充值 → 5️⃣ 在TopVanAI中配置
```

**操作**: 请按照下面的完整教程从第一步开始操作

## 🔧 DeepSeek 配置教程

### 第一步:注册DeepSeek账号

1. **打开网站**: 在浏览器输入 `https://platform.deepseek.com`

![DeepSeek官网首页](./assets/001_05f1e0d5e8.png)

2. **点击注册**: 找到"登录/注册"按钮,点击进入

![点击注册按钮](./assets/002_5332ff5a91.png)

3. **填写注册信息**:
   - 可以选择手机号、邮箱或微信扫码注册
   - 输入手机号/邮箱或扫码
   - 点击"获取验证码"
   - 输入收到的验证码
   - 设置登录密码

![输入注册信息](./assets/003_a10ee181a1.png)

4. **完成注册**: 点击"注册"按钮完成

### 第二步:实名认证

根据国家规定,使用AI服务需要完成实名认证。

1. **进入认证页面**: 登录后系统会提示实名认证,或点击左下角"个人信息"进入实名认证

![实名认证入口](./assets/004_6a96d93467.png)
![实名认证页面](./assets/005_4807b0ecd6.png)

2. **填写认证信息**:
   - 输入真实姓名
   - 输入身份证号码
   - 上传身份证正反面照片

![填写身份信息](./assets/006_e2e741154d.png)

3. **等待审核**: 提交后等待系统审核,通常几分钟内完成

### 第三步:获取API Key

API Key是访问模型服务的密钥,请妥善保管。

1. **进入API管理**: 点击左侧菜单的"API Keys"

![API Keys菜单](./assets/007_dd1d35cf94.png)

2. **创建新密钥**:
   - 点击"创建API Key"按钮
   - 给密钥命名(如"TopVanAI")
   - 点击"创建"

![创建API Key](./assets/008_278c1e76ce.png)
![创建API Key确认](./assets/009_1f420176ec.png)

3. **复制并保存密钥**:
   - 密钥创建后会显示一长串字符,格式类似 `sk-xxxxxxxxxxxxx`
   - 点击"复制"按钮
   - ⚠️ **重要**: 密钥只显示一次,请立即保存

![复制API Key](./assets/010_5bc6fba901.png)

💾 **保存方式参考**:
- 保存到电脑记事本/备忘录
- 发送到微信"文件传输助手"
- 保存到手机备忘录

### 第四步:账户充值

1. **进入充值页面**: 点击左侧菜单"账户充值"

![充值菜单](./assets/011_a8b7d7c0ae.png)

2. **选择充值金额**:
   - 查看最低充值金额要求
   - 选择合适的充值金额
   - 支持支付宝、微信支付

![选择充值金额](./assets/012_9dd643f4da.png)

3. **完成支付**:
   - 选择支付方式
   - 扫码支付
   - 支付成功后余额到账

![支付页面](./assets/013_94f0c7a3d0.png)

### 第五步:在TopVanAI中配置

1. **打开TopVanAI设置**:
   - 启动TopVanAI
   - 点击右上角"设置"图标(齿轮⚙️)
   - 点击左侧"模型"选项

![TopVanAI设置入口](./assets/014_e6dd88d470.png)
![TopVanAI模型设置](./assets/015_8f53d5ee5c.png)

2. **找到DeepSeek**: 在模型列表中找到"DeepSeek"

![模型列表](./assets/016_963053812d.png)

3. **填写配置信息**:
   - **启用开关**: 点击开关按钮启用
   - **API Key**: 粘贴之前复制的密钥
   - **API Base URL**: 使用默认URL即可
   - **模型选择**: 根据需要选择可用模型

![填写DeepSeek配置](./assets/017_e5c5bbb77e.png)

4. **测试连接**:
   - 点击"测试连接"按钮
   - 检查连接状态
   - 如连接失败,请检查:
     - API Key是否完整(无多余空格)
     - Base URL是否正确
     - 账户余额是否充足

![测试连接](./assets/018_6ed9fc2fa0.png)

5. **保存配置**: 测试成功后点击"保存"按钮

### 开始使用

配置完成后:
1. 回到聊天界面
2. 在模型选择器中选择"DeepSeek"
3. 即可开始对话

![选择模型](./assets/019_6fbb688e30.png)

## ☁️ 阿里通义千问 配置教程

### 第一步:注册阿里云账号

如果已有淘宝/支付宝账号,可以直接登录。

1. **打开阿里云官网**: 浏览器输入 `https://www.aliyun.com`

   ![阿里云官网](./assets/029_0c574e2cee.png)

2. **点击注册**: 点击右上角"免费注册"或"登录"按钮

   - 已有账号:点击"登录"扫码登录
   - 新用户:点击"免费注册"用手机号注册

   ![阿里云注册登录](./assets/030_5107dc447b.png)

3. **完成注册**:
   - 输入手机号
   - 获取并输入验证码
   - 设置登录密码
   - 点击"注册"完成

⚠️ **重要提示**: 阿里云账号注册后需要完成实名认证才能使用百炼服务。

### 第二步:完成实名认证

注册成功后,系统会引导完成实名认证。

1. **进入认证页面**:
   - 登录后可能自动跳转到"欢迎来到我的阿里云"页面
   - 或在右上角头像菜单中找到"实名认证"

2. **填写认证信息**:
   - 输入真实姓名
   - 输入身份证号码
   - 上传身份证正反面照片

   ![实名认证](./assets/004_6a96d93467.png)

3. **等待审核**: 提交后等待系统审核,通常几分钟内完成

   ✅ 认证成功后,"实名认证"卡片会显示"已完成"

### 第三步:开通阿里云百炼服务

**重要**: 通义千问使用的是阿里云百炼(Bailian)平台,需要先开通服务。

1. **打开百炼控制台**:
   - 在浏览器输入: `https://bailian.console.aliyun.com`
   - 或访问: `https://www.aliyun.com/product/bailian` 点击"控制台"

   ![百炼控制台入口](./assets/031_1f2a827bf3.png)

2. **首次开通服务**:
   - 系统可能会提示"欢迎使用阿里云百炼"
   - 找到"免费试用"或"立即开通"按钮
   - 点击开通并同意服务协议

   ![开通百炼服务](./assets/032_c40fc31aec.png)

3. **进入控制台首页**:
   - 顶部菜单包括"模型服务"、"应用开发"等
   - 左侧有导航菜单
   - 开通成功后可以创建API Key

### 第四步:创建API Key

服务开通后,就可以创建API Key了。

1. **进入API Key管理页面**:
   - 方法一:在百炼控制台左侧菜单找到"API Key"
   - 方法二:直接访问: `https://bailian.console.aliyun.com/?tab=model#/api-key`

   ![API Key菜单](./assets/033_e08b7bc944.png)

2. **创建新密钥**:
   - 点击"创建API Key"或"创建"按钮
   - 在弹窗中配置:
     - **归属账号**: 建议选择主账号(显示为纯数字ID)
     - **归属业务空间**: 建议选择"默认业务空间"
   - 点击"确定"完成创建

   ![创建API Key弹窗](./assets/034_4a8a6f937e.png)

3. **复制并保存API Key**:
   - 密钥创建后会显示一长串字符,格式类似 `sk-xxxxxxxxxxxxxxxx`
   - 点击"复制"按钮或点击眼睛图标显示密钥
   - ⚠️ **重要**: 密钥只显示一次,请立即保存到安全位置

   ![复制API Key](./assets/035_2dadcc7947.png)

💾 **保存方式参考**:
- 保存到电脑记事本/备忘录
- 发送到微信"文件传输助手"
- 保存到密码管理器
- 不要上传到Git仓库或公开场合

### 第五步:查看免费额度(新用户专享)

新用户首次开通阿里云百炼会获得免费额度。

1. **查看免费额度**:
   - 在百炼控制台点击"模型用量"
   - 或访问: `https://bailian.console.aliyun.com/?tab=model#/model-usage/free-quota`
   - 点击"免费额度"页签

   ![查看免费额度](./assets/036_05f3080eef.png)

2. **额度详情**:
   - 显示格式: `362,917/1,000,000` (已用/总计)
   - 查看过期时间
   - 不同模型的免费额度独立计算

3. **开启用完即停功能**(推荐):
   - 在免费额度页面找到目标模型
   - 打开"免费额度用完即停"开关
   - 额度耗尽后自动停止,避免扣费

   ![用完即停开关](./assets/037_d9b53dc4d3.png)

> ⚠️ 免费额度每小时更新,非实时数据。

### 第六步:账户充值

免费额度用完后需要充值才能继续使用。

1. **进入充值页面**:
   - 在百炼控制台右上角点击"费用"或"充值"
   - 或访问阿里云费用中心: `https://usercenter2.aliyun.com/home`

   ![充值入口](./assets/038_2d7d0fae14.png)

2. **选择充值金额**:
   - 查看最低充值金额要求
   - 选择合适的充值金额
   - 支持支付宝、微信支付、网银、对公转账

   ![选择充值金额](./assets/039_c0838fb7ef.png)

3. **完成支付**:
   - 扫码或在线支付
   - 支付成功后余额实时到账(可能有几分钟延迟)

   ![支付成功](./assets/040_83d7111aff.png)

### 第七步:在TopVanAI中配置

完成以上所有步骤后,就可以在TopVanAI中使用通义千问了。

1. **打开TopVanAI设置**:
   - 启动TopVanAI应用
   - 点击右上角"设置"图标(齿轮⚙️)
   - 点击左侧"模型"选项

   ![TopVanAI设置入口](./assets/014_e6dd88d470.png)
   ![TopVanAI模型设置](./assets/015_8f53d5ee5c.png)

2. **找到通义千问**: 在模型列表中找到"Qwen"或"通义千问"

   ![模型列表](./assets/041_1f97e3864f.png)

3. **填写配置信息**:
   - **启用开关**: 点击开关按钮启用
   - **API Key**: 粘贴之前复制的阿里云API Key
   - **API Base URL**: 使用默认URL即可
   - **模型选择**: 选择需要使用的模型(如qwen-plus、qwen-max等)

   ![填写通义千问配置](./assets/042_6cd4ea5a7c.png)

4. **测试连接**:
   - 点击"测试连接"按钮
   - 检查连接状态是否显示"成功"
   - 如连接失败,请检查:
     - API Key是否完整(无多余空格)
     - Base URL是否正确
     - 账户余额或免费额度是否充足
     - 网络连接是否正常

   ![测试连接](./assets/043_4333f510e4.png)

5. **保存配置**: 测试成功后点击"保存"按钮

### 开始使用

配置完成后:

1. 回到TopVanAI聊天界面
2. 在模型选择器中选择"通义千问"或"Qwen"
3. 即可开始对话

![选择通义千问模型](./assets/044_c3bb171363.png)

## ❓ 常见问题

### 1. 我已经有账号了,怎么快速配置?

如果您已经在服务商平台注册过账号:

**步骤1**: 登录服务商平台,确认以下信息:
- ✅ 是否有可用的API Key(如果没有,创建一个)
- ✅ 账户余额是否充足(如果不足,充值)

**步骤2**: 打开TopVanAI设置 → 模型 → 找到对应服务商

**步骤3**: 填写配置:
- 启用开关
- 粘贴API Key
- 填写API Base URL(参考对应服务商教程中的URL)
- 测试连接并保存

**快速查找**: 返回[配置流程概览](#配置流程概览)部分,使用快速跳转链接直达配置步骤。

### 2. API Key提示无效怎么办?

**检查步骤**:
- 检查复制时是否包含多余空格
- 确认API Key是否完整
- 确认账户余额是否充足
- 检查API Base URL是否正确

**解决方法**:
1. 重新复制粘贴API Key
2. 检查账户余额
3. 确认Base URL格式正确(无多余空格或斜杠)

### 3. 测试连接失败怎么办?

**常见错误及处理**:

| 错误提示 | 可能原因 | 处理方法 |
|---------|---------|---------|
| "API Key无效" | 密钥错误或不完整 | 重新复制完整密钥 |
| "余额不足" | 账户余额不足 | 充值账户 |
| "网络错误" | 网络连接问题 | 检查网络连接 |
| "服务不可用" | 服务商维护 | 访问服务商官网查看状态 |

### 4. 如何查看消费记录?

各平台查看方式:

**DeepSeek**:
- 登录后台 → 左侧菜单"消费明细"

**智谱AI**:
- 控制台 → "账户余额" → "消费记录"

**通义千问**:
- 阿里云控制台 → "费用" → "消费明细"

**Moonshot**:
- 费用中心 → "消费记录"

### 5. 多个模型可以同时配置吗?

可以。TopVanAI支持同时配置多个模型服务商,您可以:
- 在对话中切换不同模型
- 同时使用多个服务商的服务
- 根据需要选择使用的模型

### 6. API Key丢失怎么办?

**各平台处理方式**:

| 平台 | 是否可重新查看 | 处理方法 |
|------|--------------|---------|
| DeepSeek | 不可以 | 删除旧密钥,创建新密钥 |
| 智谱AI | 不可以 | 创建新密钥 |
| 通义千问 | 可以 | 在API-KEY管理页面查看 |
| Moonshot | 不可以 | 创建新密钥 |
| MiniMax | 可以 | 在应用详情中查看 |

建议:将所有API Key保存在安全的地方

### 7. 不同服务商如何选择?

各服务商提供不同的模型和服务,您可以:
1. 查看各服务商官网了解其模型特点
2. 配置多个服务商进行对比测试
3. 根据实际使用效果选择

### 8. 遇到其他问题怎么办?

**获取帮助的途径**:

1. **查看服务商官方文档**:
   - DeepSeek: `https://platform.deepseek.com/docs`
   - 智谱AI: `https://open.bigmodel.cn/dev/howuse/introduction`
   - 通义千问: `https://help.aliyun.com/zh/dashscope`
   - Moonshot: `https://platform.moonshot.cn/docs`
   - MiniMax: `https://www.minimaxi.com/document`
